● Factor 1 of 3
Knowledge Factor
Enter Your Password
Your password is never stored in plaintext. It was hashed with Argon2ID — a memory-hard algorithm designed to resist brute-force attacks.
What's happening: The server computes Argon2ID(password + salt) and compares it to the stored hash. If it matches, a signed JWT with status pending_f2 is issued — no session stored server-side.
● Factor 2 of 3
Possession Factor
Enter Your TOTP Code
Open Google Authenticator (or Apple Passwords) and enter the 6-digit code shown for your account.
What's happening: The server runs TOTP (RFC 6238) — HMAC-SHA1 over the current Unix time slot (30 s window). The code is valid for ±30 s to absorb clock skew. No code is ever reusable.
